# Predictions-of-Ratings-of-an-Anime( Regression)


A streaming company named streams web series and movies for a worldwide. Every content on their portal is rated by the viewers, and the portal also provides other information for the content like the number of people who have watched it, the number of people who want to watch it, duration of an episode, the number of episodes, etc. So here we need to identify the important factors that are used to predict the ratings and we would build a predictive model to predict the rating on an animes.
